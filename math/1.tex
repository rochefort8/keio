\textbf{第1問}
\begin{enumerate}
\item $\sqrt{50}-3\sqrt{8}+\sqrt{18}$ を簡単にせよ。
\item $\dfrac{4}{3-\sqrt{2}}$ を有理化せよ。
\end{enumerate}

\vspace{0.5em}
\textbf{第2問}

$f(x)=x^2-6x+13$ とする。
\begin{enumerate}
\item 平方完成せよ。
\item 最小値とそのときの $x$ を求めよ。
\end{enumerate}

\vspace{0.5em}
\textbf{第3問}

$|2x-3|<5$ を解け。

\vspace{0.5em}
\textbf{第4問}

赤玉3個、白玉2個から同時に2個取り出す。
\begin{enumerate}
\item 2個とも赤である確率を求めよ。
\item 少なくとも1個が白である確率を求めよ。
\end{enumerate}

\vspace{0.5em}
\textbf{第5問}

$a_1=2,\ a_6=17$ の等差数列とする。
\begin{enumerate}
\item 公差を求めよ。
\item 一般項 $a_n$ を求めよ。
\item $\displaystyle \sum_{k=1}^{10} a_k$ を求めよ。
\end{enumerate}

\vspace{0.5em}
\textbf{第6問}

データ $3,4,6,7,10$ について、
\begin{enumerate}
\item 平均を求めよ。
\item 分散（母分散）を求めよ。
\end{enumerate}
